Extensiones para VSCode

- Python: extensión fundamental para edición, debugueo y otras cosas. Instala automáticamente otras extensiones complementarias: Python Debugger, Python Environments y Pylance.
- Prettier: otorga estilo al archivo que estés editando, así es más fácil de ver las cosas.
- Pytohn Texs Explorer: permite realizar pruebas mientras vas creando la aplicación. Añadir una vez que la estructura ya esté hecha.
- GitLens: estar al tanto de cambios e historial de los archivos.
- Docker: si la apliación estará contenida en un contenedor. Útil para desarrollo local. Añadir una vez que la estructura ya esté hecha.

Flujos de trabajo de VSCode

- Ambientes: venv o pipenv.
- Corregir código: flake8 o pylint o ruff.
